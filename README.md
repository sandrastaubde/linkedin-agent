## 🤖 Was kann der Agent?

### **Research Agent**
- Sammelt Trends in Ihren 3 Hauptthemen
- Analysiert Content-Angles
- Lernt Ihren Schreibstil aus Beispielen

### **Content Agent**
- Generiert 5-7 LinkedIn Posts
- In Ihrer Brand Voice
- LinkedIn-optimiert (3000 Zeichen Limit)
- Passt zu Ihren Hauptthemen

### **Learning System**
- Lernt aus Ihren Approvals/Rejects
- Wird mit jeder Entscheidung besser
- Passt Vorschläge an Ihre Präferenzen an

## 🎯 Workflow

1. **Onboarding** (einmalig, 3 Min)
   - Profil eingeben
   - 3 Hauptthemen wählen
   - Schreibproben hochladen

2. **Content generieren** (30-60 Sek)
   - "Neue Vorschläge generieren" tippen
   - Agent erstellt 5 Posts

3. **Bewerten** (10 Sek pro Post)
   - ✅ Approve (Agent lernt: Das gefällt mir!)
   - ❌ Reject (Agent lernt: Das vermeiden!)

4. **Besser werden** (automatisch)
   - Nach 10-20 Entscheidungen
   - Agent kennt Ihre Präferenzen
   - Vorschläge werden immer besser!
